1.3 查看好友信息("/ajax/friend_info")

请求格式

}
回复格式
{
    "f_icon",    
    "f_name",    
    "recentPlay":[{
        "gid",

    }]
}

1.4 验证token有效期("/ajax/token_expires")

请求格式
{
    "token",    
    "uid"    
}
回复格式
{
    "ret"    1 表示有效 0表示无效，需要重新登录
}
